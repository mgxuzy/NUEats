# NUEats

Food ordering application built with Swing.

## Prerequisites

* JDK 25

## Usage

### Windows

```pwsh  
.\gradlew.bat run  
```  

### Unix-based OS

```sh
./gradlew run  
```  

## Architecture

Modular design following clean architecture principles, featuring a custom Pluggable Look and Feel (PLAF) and decoupled
domain logic.
