# NUEats

A food ordering application built with Java Swing.

## Prerequisites

* Install [JDK 25](https://adoptium.net/temurin/releases/?version=25).

## Usage

### Windows

```powershell
.\gradlew run  
```  

### Unix-based OS

```shell
./gradlew run  
```  

## Architecture

Modular design following clean architecture principles, with a custom Pluggable Look-and-Feel (PLAF) and decoupled domain logic.
